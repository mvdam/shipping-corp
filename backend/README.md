# Shipping Corp Backend

### Install

Install the dependencies:

`npm install`

### Run

To run the project in development mode:

`npm run start:dev`

Exposes the Backend at `https://localhost:9000`.

### Build

To build the project:

`npm run build`

### Test

To test the project:

`npm run test`

### Endpoints

- `/container/:id`
- `/container/:id/parcels`
- `/container/:id/parcelsByDepartment`
- `/department/all`
